有时候我们会想让某些事情「重复做好多次」，比如让角色执行一连串重复的动作，批量更改节点的属性或是一下子生成一大堆东西
这时候，**循环语句**就派上用场了

本章将带你认识两种常见的循环方式：

* **while** —— 只要条件成立，就一直转圈圈，直到不满足为

* **for** —— 适合次数明确的循环，用来遍历、计数、干活都很方便

除此之外，我们还会认识两个跳转语句：

* **break** —— 让你立刻跳出循环

* **continue** —— 让你跳过当前这轮，直接进入下一次迭代
