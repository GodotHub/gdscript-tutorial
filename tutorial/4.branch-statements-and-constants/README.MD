# 分支语句与常量

在游戏中，玩家做出的每一个选择，都会推动故事走向不同的发展方向
而在编程中，如果我们也想根据不同的情况来“走不同的路”，那就少不了 **分支语句**

本章将带你了解分支语句的基本用法，以及如何根据「条件是否成立」来决定执行哪一段代码
我们还会介绍一个常搭配使用的好朋友——**常量和枚举**，它可以让你的分支判断更清晰、更安全
